# Правила составления заявок

## Содержание и тон

- Использовать официально-деловой тон
- Подробно описывать одну наблюдаемую проблему с признаками и последствиями
- Указывать практические последствия, только если они известны из ввода
- Добавлять раздел «Прошу:» с нумерованными пунктами
- Преобразовывать эмоции в наблюдаемые факты

## Ограничения

- Не придумывать место, причину, виновника или повреждения
- Не добавлять неподтверждённые обвинения
- Не добавлять вымышленные даты, номера договоров и актов
- Не создавать отдельный раздел «Правовое обоснование»
- Не просить предоставить письменный ответ по умолчанию

## Пример

**Ввод жителя:**
«Калитка во двор сломана, не закрывается, магнит болтается. В подъезд теперь кто угодно может зайти, страшно. Сделайте уже что-нибудь!»

**Наблюдаемый факт вместо эмоции:**
«В подъезде сохраняется резкий неприятный запах» вместо «В подъезде опять невозможно находиться»

**Размытая просьба:**
«Сделайте уже что-нибудь»

**Выполнимая просьба:**
«Прошу проверить источник запаха и устранить причину»

**Неподтверждённое объяснение:**
«Трубу точно повредили рабочие»

**Безопасная формулировка:**
«На трубе заметна течь. Прошу провести осмотр и устранить неисправность»
